# Penguin Game

This project was developed as part of a university programming course to practice modular programming, dynamic memory management, and multi-file project architecture in C.

---

## Features

- Turn-based penguin board game
- Multiple players support
- Modular game architecture
- Save game functionality
- Terminal interface

---

## Technologies

- C Programming Language
- GCC / Clang
- Modular Programming (.c / .h separation)

---

## How to Compile & Run

Using gcc:

```bash
gcc main.c board.c game.c players.c -o Penguin-Game
./Penguin-Game
```

---

## Learning Goals

This project focuses on:

- Multi-file C project structure
- Memory management
- Game logic implementation
- Separation of concerns
